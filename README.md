# TalentScout — Hiring Assistant (Gradio)

**Project**: AI/ML Intern Assignment — Hiring Assistant chatbot  
**Purpose**: Initial screening assistant that collects candidate info, generates technical questions based on candidate tech stacks, and maintains conversation context.

_Source assignment used for requirements: included in repo._ :contentReference[oaicite:1]{index=1}

---

## Features

- Gradio UI for conversational flow
- Collects: Full name, email, phone, years of experience, desired position, location, tech stack
- Generates 3–5 targeted questions per technology listed
- Maintains conversation context
- Fallback responses and exit keywords
- Optional: sentiment analysis & multilingual support (detect & translate)
- Dockerfile + GitHub Actions workflow included
